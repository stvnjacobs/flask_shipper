About

This is currently a side project that aims to be a user interface for the EasyPost shipping api. It began as a way to scractch an itch after watching a co-worker get repeatedly frustrated with USPS shipping, and another always in need of shipping quotes. An efficient way to interact with many carriers at once could only be beneficial.

Also, this project is my first attempt to add a web interface and real user input to a Python application, both of which I get help from the Flask microframework. The main goal is simplicity for the user.

Roadmap & Ideas:

  • User accounts to allow for storing freight account information. Necessary for the use of negotiated rates with EasyPost's large list of connected shipping services
  • Ability to purchase and print labels
  • Tracking packages
  • Warnings, concerns, can't ship away sister or cat, etc.

Notes

  • USPS Rates Provided. These are Commercial Plus Pricing provided by easypost.
  • UPS Rates Provided These are non-negotiated rates, equal to those from a newly created UPS account, since thats what they are!
